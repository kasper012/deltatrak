Applications: Transport, Cold Storage, Processing Facilities

This single-use in-transit logger is accepted by most receivers and is the ideal cold chain management tool for tracking temperature during transportation of perishable products. It can be used on all outbound loads, so suppliers are able to consolidate their temperature recorders to one model. Using the FlashLink® VU In-Transit Logger means there is no need to stock various brands in the shipping office, making inventory management easy, and eliminating the problem of running out of any one particular type of logger for a specific receiver.

Model 20879 is factory configured with alarm settings to alert receivers with a flashing red LED when temperature during transit has gone above 41°F/5°C or below 33°F/0.6°C, and has a 30 minute delay start and a 30 minute alarm delay. The patented Shadow Log™ feature guarantees data, even if the shipper forgets to start the unit. The loggers are downloaded using free FlashLink Data Manager software, and reports are saved in PDF format to email and archive for record keeping, traceability and to help comply with HACCP and FSMA (Food Safety Modernization Act) requirements.

Packaging

Each FlashLink® VU Logger is mounted on a bright green shipping card. This makes it easy to find when a load arrives and enables receivers to download the trip report immediately. The card has printed instructions in English and Spanish, for shippers to start the logger and receivers to download the unit. A diagram identifies and describes the various icons that can be displayed on the LCD.

Software

FlashLink Data Manager 6.7.2 + 8-Pin Adapter

Last Revised - 06/14/2021

FlashLink Data Manager is a simple program used to create a locked PDF file with data downloaded from any FlashLink data logger. Once installed on a Windows PC, the program runs automatically when a data logger is connected via the USB adapter. The PDF is then created, saved, and displayed on screen automatically. These PDF files can easily be archived and emailed.

Connecting a FlashLink 8-pin data logger to a USB port requires the use of the DeltaTrak 8-pin to USB adapter. FlashLink Data Manager has been optimized to work with the DeltaTrak 8-Pin to USB High Speed Adapter and is available for ordering below.
